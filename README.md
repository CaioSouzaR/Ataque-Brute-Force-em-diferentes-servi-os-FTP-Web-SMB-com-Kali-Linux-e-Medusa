# Ataque-Brute-Force-em-diferentes-servi-os-FTP-Web-SMB-com-Kali-Linux-e-Medusa
Desafio Dio.me simulando um Ataque de Brute Force de Senhas com Medusa e Kali Linux, utilizando diferentes serviços (FTP, Web, SMB) em ambiente controlado, para simular cenários de ataque de força bruta e exercitar medidas de prevenção.
